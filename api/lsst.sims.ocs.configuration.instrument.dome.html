lsst.sims.ocs.configuration.instrument.dome module¶

class lsst.sims.ocs.configuration.instrument.dome.Dome[source]¶

Bases: lsst.pex.config.config.Config

Configuration of the LSST Dome.

!Allocate a new Config object.

In order to ensure that all Config object are always in a proper state when handed to users or to derived Config classes, some attributes are handled at allocation time rather than at initialization

This ensures that even if a derived Config class implements __init__, the author does not need to be concerned about when or even if he should call the base Config.__init__

setDefaults()[source]¶

Set defaults for the LSST Dome.

altitude_accel¶

Maximum acceleration (units=degrees/second**2) of dome altitude movement.

altitude_decel¶

Maximum deceleration (units=degrees/second**2) of dome altitude movement.

altitude_maxspeed¶

Maximum speed (units=degrees/second) of dome altitude movement.

azimuth_accel¶

Maximum acceleration (units=degrees/second**2) of dome azimuth movement.

azimuth_decel¶

Maximum deceleration (units=degrees/second**2) of dome azimuth movement.

azimuth_maxspeed¶

Maximum speed (units=degrees/second) of dome azimuth movement.

settle_time¶

Times (units=seconds) for the dome to settle after stopping.
